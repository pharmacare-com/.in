# 📱 PharmaCare - Premium Medical Inventory & Billing App

PharmaCare is a premium, offline-first medical store inventory, billing, and khata management application. It features real-time billing counters, a bluetooth thermal invoice simulator, a clean 10-day trash retention policy, and high-fidelity mobile UI simulators.

## 🚀 Fully Automatic GitHub Auto-Sync & Deployment Workflow
हमने GitHub के लिए एक पूरी तरह से स्वचालित (fully automatic) CI/CD Workflow तैयार किया है, जो आपकी साइट को हमेशा लाइव और सिंक रखता है। 

Whenever you push your code to GitHub on the `main` or `master` branch, the automatic workflow will build and sync your application to **GitHub Pages** instantly.

### 🛠️ How to Enable Auto-Sync / इसे कैसे सेटअप करें:

#### हिन्दी में (In Hindi):
1. इस प्रोजेक्ट को अपने **GitHub** अकाउंट पर अपलोड (push/import) करें।
2. अपने GitHub Repository में जाएँ और ऊपर **Settings** टैब पर क्लिक करें।
3. लेफ्ट साइडबार में **Pages** विकल्प पर जाएँ।
4. **Build and deployment** सेक्शन के अंदर, **Source** ड्रॉपडाउन को **Deploy from a branch** से बदलकर **GitHub Actions** सिलेक्ट करें।
5. अब जब भी आप नया अपडेट पुश करेंगे, GitHub Actions अपने आप प्रोजेक्ट को बिल्ड करके ऑनलाइन सिंक और डिप्लॉय कर देगा!

#### In English:
1. Push/Import this repository to your **GitHub** account.
2. Navigate to your GitHub Repository **Settings** tab.
3. Click on **Pages** in the left sidebar menu.
4. Under **Build and deployment**, change the **Source** dropdown selection to **GitHub Actions**.
5. That's it! Every future commit to the `main` or `master` branch will trigger an automated build and deploy the application to your live URL.
